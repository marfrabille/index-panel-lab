# index-panel-lab

## 项目简介

`index-panel-lab` 是一个用于归档和发布多个独立 HTML 页面的仓库。  
本仓库不针对任何特定域名或网站，仅作为通用 HTML 页面的存放与展示空间。  
适合存放各类实验性、示范性或存档性质的网页文件。

## 目录说明

```
/
├── index.html          # 主入口页面（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── example1.html
│   └── example2.html
├── assets/             # 公共资源（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- **pages/**：主要的 HTML 页面归档目录，每个页面相互独立。
- **assets/**：存放页面共享的静态资源，按类型分目录管理。

## 页面归档说明

- 所有归档 HTML 页面均为独立文件，不依赖外部特定服务或 API。
- 页面内容保持简洁，便于直接通过浏览器打开查看。
- 归档文件命名建议使用英文小写、连字符分隔，如 `demo-tool.html`。
- 如需添加新页面，请将文件放入 `pages/` 目录，并确保引用资源路径正确。

## 维护说明

- 本仓库由维护者不定期更新，添加新的 HTML 页面或调整现有内容。
- 提交新页面时，请保持目录结构清晰，避免重复或冲突的文件名。
- 欢迎提交 Pull Request 或 Issue 讨论页面归档相关建议。
- 仓库不提供在线演示或托管服务，仅作为静态源码存档。
